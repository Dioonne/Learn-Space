# Learn-Space
An educational web portal exploring deep space, the International Space Station, and exoplanetary data. Built with a glassmorphism UI and responsive design
# Learn Space: An Interactive Portal

Welcome to **Learn Space**, a web-based educational platform designed to bring the mysteries of the universe closer to Earth. This project provides a curated look at the International Space Station (ISS) and catalogs significant exoplanets outside our solar system.

# Key Features
*   **ISS Explorer:** A dedicated section for the International Space Station with direct links to the NASA Image Library.
*   **Exoplanet Database:** A comprehensive data table featuring vital statistics (distance, host star, mass) for planets like Proxima Centauri b and TRAPPIST-1e.
*   **Glassmorphism UI:** Features a modern `glass-nav` navigation bar and clean, organized sections for high readability.
*   **Educational Resources:** Integrated references from NASA, Caltech, and the Planetary Habitability Laboratory.

# Technology Stack
*   **HTML5:** Semantic structure including forms and data tables.
*   **CSS3:** Custom styles located in `/css/styles.css` (includes responsive layouts).
*   **Assets:** Organized media storage for astronomical imagery.
